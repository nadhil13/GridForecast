# Security Policy

## Supported Versions

We strive to ensure our software is secure and reliable. Below is the list of versions currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for us. If you discover any security-related issues or vulnerabilities, please **do not** report them through public GitHub issues. 

Instead, please contact the maintainer directly. We will treat your report with the utmost confidentiality and work to resolve the issue as quickly as possible. We appreciate your efforts to responsibly disclose your findings.
